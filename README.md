# ✈️ Flight Delay Analysis 2024

Exploratory data analysis of U.S. flight delays in 2024, uncovering which airlines and airports experience the highest average arrival delays, and how delay durations are distributed across flights.

## 📊 Project Overview

This project analyzes a dataset of approximately 539,000 flight records from 2024 to answer key questions about flight delay patterns:

- Which **airlines** have the highest average arrival delays?
- Which **airports** have the highest average arrival delays?
- How are **arrival delays distributed** across all flights?

## 🗂️ Dataset

- **Source:** [Flight Delay Dataset — 2024 (Kaggle)](https://www.kaggle.com/datasets/hrishitpatil/flight-data-2024)
- **Size:** ~539,000 records
- **Note:** The raw dataset is not included in this repository due to its size. Download it directly from the Kaggle link above to reproduce the analysis.
